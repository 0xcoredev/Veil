# Veil тАФ Level 4 Submission

## Project Overview
**Veil** is a privacy-first anonymous chat application built on Stellar. It features end-to-end encrypted messaging, wallet-based authentication, token-gated communities, DAO governance, and micropayments.

**Tagline:** Speak freely. Stay veiled. Powered by Stellar.

## 8. Technical Architecture

### Tech Stack
| Layer | Technology |
|---|---|
| Frontend | Next.js 16, React 19, TypeScript, Tailwind CSS 4 |
| UI | shadcn/ui (Radix UI), Lucide icons |
| Auth | SEP-10 (Stellar auth), Freighter wallet |
| Database | Supabase (PostgreSQL) |
| Real-time | Supabase Realtime |
| Blockchain | Stellar (Soroban smart contracts in Rust) |
| Encryption | Web Crypto API (X25519 + AES-GCM) |
| Key Storage | IndexedDB |
| Wallet | @stellar/freighter-api |
| Payments | Stellar SDK (XLM, USDC) |
| Hosting | Vercel |
| Analytics | Vercel Analytics + Speed Insights |

### Smart Contracts (Deployed on Testnet)
1. **Token Gate** тАФ Manages token-gated room access
2. **Governance** тАФ DAO-based moderation voting
3. **Reputation** тАФ Non-transferable reputation tracking

### Key Features
- End-to-end encrypted messaging (X25519 + AES-GCM)
- Zero-knowledge wallet authentication (SEP-10)
- Token-gated community rooms
- DAO governance for moderation
- Micropayments and tipping
- PWA support
- Dark/light theme
- Mobile responsive UI
